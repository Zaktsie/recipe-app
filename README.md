# Recipe App | Master the Kitchen with Ease

A professional, feature-rich recipe discovery platform built with **Nuxt 3**, **Tailwind CSS**, and **TypeScript**. This application allows users to explore thousands of recipes, search by ingredients or names, and manage their personal favorites.

## 🚀 Key Features

- **🔍 Smart Search & Filtering:** Find recipes in real-time by name or filter through various categories and cuisines.
- **❤️ Favorites System:** Save your preferred recipes to your personal collection with persistent browser storage (`localStorage`).
- **👨‍🍳 Interactive Cooking Mode:** Check off ingredients as you prep and mark instruction steps as "complete" while you cook.
- **✨ Premium UI/UX:**
  - **Skeleton Loaders:** Smooth, animated loading states for better perceived performance.
  - **Difficulty Badges:** Color-coded badges (Easy, Medium, Hard) for quick scanning.
  - **Clickable UI:** Every recipe card image and button is fully interactive.
- **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **🛡️ SEO Optimized:** Dynamic meta tags and titles for every page to improve search visibility.

## 🛠️ Tech Stack

- **Framework:** [Nuxt 3](https://nuxt.com/) (with Nuxt 4 compatibility mode)
- **Styling:** [Tailwind CSS](https://tailwindcss.nuxtjs.org/)
- **State Management:** [Vue Composables](https://vuejs.org/guide/reusability/composables.html) (with `useState`)
- **API Support:** [DummyJSON Recipes API](https://dummyjson.com/docs/recipes)
- **Icons:** [Nuxt Icon](https://nuxt.com/modules/icon)
- **Images:** [Nuxt Image](https://image.nuxt.com/) (optimized WebP format)
- **Typography:** Montserrat (via Google Fonts)

## 📦 Setup & Installation

Make sure to install dependencies:

```bash
# npm
npm install
```

## 🛠️ Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## 🏗️ Production Build

Build the application for production:

```bash
# npm
npm run build
```

Locally preview the production build:

```bash
# npm
npm run preview
```
